This directory contains TypeScript type definitions for the application:

- `next-env.d.ts`: Next.js environment type declarations
- `api.ts`: Types for API requests and responses
- `database.ts`: Database model and schema types
- `resume.ts`: Resume-related type definitions
- `user.ts`: User model and authentication types
- `firebase.ts`: Firebase service type extensions
- `components.ts`: React component prop interfaces
- `global.d.ts`: Global type declarations and modules
