# Configuration Directory Structure

The `config` directory contains various configuration files used throughout the application:

- `fonts.ts` - Font configurations using next/font
- `postcss.config.js` - PostCSS configuration settings
- `tailwind.config.js` - Tailwind CSS framework configuration
- `firebase.config.ts` - Firebase client-side configurations
- `constants.ts` - Application-wide constants including API URLs and theme values
- `site.config.ts` - Site metadata and navigation configuration
- `analytics.config.ts` - Analytics setup and configurations
